Production-ready deployment

This app is built to run on Render using Docker. The same API powers both the browser UI and programmatic clients.

API Endpoints

  • GET / - dashboard UI
  • GET /health - health & service info
  • POST /predict - single record inference
  • POST /batch-predict - CSV upload inference
  • GET /valid-values - valid categorical values
  • GET /docs - Swagger docs

Quick tips

  • Use the browser form for fast testing.
  • Use /docs for schema examples.
  • Upload CSV files with original UCI Adult headers.
